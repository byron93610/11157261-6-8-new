互動學習遊戲：《台灣留學日記》（Annie's Taiwan Diary）
📌 關於本遊戲 (About the Game)
本遊戲是一款專為華語學習者設計的「生活模擬型文字互動遊戲（Life Simulation Text RPG）」，使用 Twine SugarCube 架構開發。

故事圍繞著剛到台灣的美國留學生安妮，透過她與台灣同學小妤的日常互動，將華語教學融入真實的台灣生活情境中。遊戲專門對接《當代中文課程》第一至三冊的語彙與語法，讓學習者在文字冒險中複習中文，同時培養跨文化適應能力。

📖 故事大綱 (Storyline)
主角安妮（Annie）是一位來自美國的留學生，帶著興奮卻又緊張的心情第一次踏上台灣的土地。面對全中文的環境與陌生的生活習慣，她感到有些不知所措。

幸好，熱心的台灣大學生小妤成為了她的最佳嚮導！在小妤的陪伴下，安妮必須獨自挑戰台灣生活中的五大日常場景。玩家將扮演安妮，透過一次次「你該怎麼做呢？」的抉擇，幫助安妮克服文化衝擊，順利從台灣的大學畢業！

🎯 核心介紹內容 (Featured Content)
本遊戲精選了 5 個外籍學生在台灣生活最常遇到、也最不可或缺的文化與語言情境：

🏫 學校：學習如何用道地且有禮貌的中文問路，以及在台灣校園中尊稱教授與長輩的社交禮儀。

🏪 超商：體驗台灣獨特的便利商店文化，挑戰結帳時店員必問的「發票載具」與「微波加熱」日常對話。

📚 圖書館：了解校園圖書館的借閱規範（憑學生證）以及台灣公共場所嚴格禁止飲食的行為準則。

🏢 宿舍：模擬集體住宿生活，學習台灣必修的「垃圾分類與資源回收」以及深夜體貼室友的社交界線。

🍧 夜市：走進台灣代表性的飲食文化，練習點手搖飲時高度客製化的「甜度冰塊（微糖微冰）」點法，並培養排隊與現金結帳的日常習慣。

🎁 玩家預期收穫 (Learning Outcomes)
完成這場文字冒險後，外籍華語學習者將獲得以下收穫：

情境式中文應用：跳脫課本的死板語法，學會台灣人在日常生活中真正使用的道地華語。

台灣文化解碼：免去現實中因文化差異產生的尷尬，提前在遊戲中內化台灣的超商、夜市、回收與公共禮儀等生活常識。

從錯誤中學習：遊戲設計了詳細的「錯題解析原因」，讓學習者在答錯時不會感到挫折，而是能立刻明白「台灣人為什麼會這樣做」，達到最佳的素養內化效果。

🎮 遊戲玩法與操作指南 (How to Play)
🕹️ 操作方式
點擊地圖自由探索：遊戲啟動後會進入「台灣生活地圖」，玩家可以自由點擊地圖上的 5 個場景標籤，沒有固定的先後順序。

三選一抉擇：進入場景後，會依序觸發 2 個任務。每個任務皆提供 3 個選項，正確答案的順序已經打亂，玩家只需點擊文字選項即可做出決定。

📊 積分與畢業機制
計分方式：答對一題得 100 分；答錯則不給分（0分）。

錯題重來機制：若在某場景答錯，系統會跳出「小妤的文化提示」說明答錯原因。玩家必須重新挑戰該場景並拿到全部分數，才能算通過該場景。

畢業結局條件：

總分達到 1000 分（全對）：解鎖隱藏的畢業典禮關卡，出現 「恭喜你畢業了」 的完美結局。

未滿 1000 分：左側狀態欄會持續顯示目前分數，並提示 「加油！快畢業了」，引導玩家繼續探索地圖直到全對為止。
